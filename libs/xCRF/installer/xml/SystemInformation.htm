  • 8 core Intel CPU Xeon E5420 @ 2.50GHz

  • 16 Gbyte RAM

  • 2x146GB mirrored disks


    • Operating System:
    SUSE Linux Enterprise Server 10


    Database server:

    2 High Availability 2-Node Cluster based on IBM p6-595 partitions (Logical Partition):
  • 1 Cluster for development purposes

  • 1 Cluster for service delivery (production) purposes

  • 2 (scalable up to 8) Virtual Processor IBM POWER6 clock 5.0 Ghz

  • Fully redundant FC connections 4Gbps

  • 2x146GB 15Krpm HDisk

  • External Storage server: Cineca Storage Area Network with SAN Volume Controller (SVC) and IBM DS8700 Enterprise Storage Server
